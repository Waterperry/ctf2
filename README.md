# CTF v2

An assortment of cybersecurity-related data science challenges I made for a CTF that BT hosted a while ago.

## Notes

In `challenges/`, each directory contains its own challenge.

Usually, a `docker compose up --build` in the directory (or a `docker built -t XYZ . && docker run -it --rm XYZ`, along with publishing the exposed ports) is all that's required to start the challenge.
